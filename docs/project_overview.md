# SalesAI Agent: Project Overview

This project aims to build an AI-driven sales assistant for a Marketing/Development Agency. The goal is to automate the initial lead qualification process through an interactive, voice-enabled chatbot that is deeply integrated with the agency's specific knowledge base.

## Core Objectives
- **Lead Qualification**: Engage potential clients and extract business requirements in under 5 minutes.
- **Deep Integration**: Train the AI exclusively on the agency's website content, services, and team structure.
- **Multimodal Interaction**: Support both Text and Speech (Real-time Voice) communication.
- **Automated Reporting**: Generate detailed summaries, task breakdowns, and cost estimates in PDF format for the agency owner.

## Key Features
1. **Intelligent Web Crawler**: A dashboard tool that scrapes the agency's website to "train" the LLM.
2. **Sales-Focused Persona**: An LLM agent optimized for sales, not general information.
3. **Voice/Text Toggle**: Seamless transition between typing and speaking.
4. **Admin Dashboard**: For monitoring conversations, managing knowledge base updates, and viewing generated leads.
5. **PDF Lead Summary**: AI-generated professional lead reports.
