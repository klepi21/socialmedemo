# SalesAI Agent Project

This repository contains the blueprints and documentation for the SalesAI Agent, a specialized chatbot designed for a marketing/development agency.

## Project Documentation

Detailed information is organized into the following documents:

1.  **[Project Overview](./docs/project_overview.md)**: Goals, core features, and project scope.
2.  **[Technical Architecture](./docs/technical_architecture.md)**: Detailed tech stack, LLM selection, and AI/Voice infrastructure.
3.  **[Implementation Plan](./docs/implementation_plan.md)**: A 5-phase roadmap from research to deployment.
4.  **[Business & Pricing Strategy](./docs/business_strategy.md)**: Guidance on how to pitch and price this project to the client.

## Quick Tech Stack Reference

- **Frontend**: Next.js 14+ (App Router)
- **AI**: OpenAI GPT-4o (Reasoning) + Firecrawl (Web Scraping)
- **Voice**: Web Speech API / OpenAI Whisper
- **Database**: Supabase (PostgreSQL + Vector)
- **PDF**: React-PDF / Puppeteer
